.. _references_settings:

Settings
========

SVG_DEFAULT_THEME
-----------------

Define the default theme of the SVG :

- dark
- light

Default value : ``dark``

SVG_DEFAULT_SIZE
----------------

Define the default size of the SVG :

- x1
- x2
- x3
- x5
- x10

Default value : ``x3``

SVG_DEFAULT_VARIANT
-------------------

Define the default variant of the SVG :

- none
- glass

Default value : ``''``
