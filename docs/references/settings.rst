.. _references_settings:

Settings
========

SVG_DEFAULT_THEME
-----------------

Define the default theme of the SVG :

- dark
- light

Default value : ``dark``

SVG_DEFAULT_WIDTH
-----------------

Default value : ``100``

SVG_DEFAULT_HEIGHT
------------------

Default value : ``100``

SVG_DEFAULT_VARIANT
-------------------

Define the default variant of the SVG :

- none
- glass

Default value : ``'glass'``
